# StockVision（智能股票技术分析平台）

> 基于 Web 的美股技术分析平台，自动计算支撑阻力位、识别技术形态

## 技术栈
Vue3, TypeScript, Vite, FastAPI, pandas, scipy, TradingView Lightweight Charts

## 核心模块
- 数据获取层：腾讯证券(主) + Yahoo(备) 自动故障转移，本地 SQLite 缓存
- 技术分析层：支撑阻力位、区间统计（多周期自适应）
- 前端展示层：K线图、搜索、自选股、深色模式

## 关键约束
- 代码在 stage_one/ 子目录，不在根目录
- 前后端算法必须 1:1 同步（支撑阻力位计算）
- 测试模式开关默认开启，生产环境需关闭
- K线周期：日K/周K/月K/季K(3M)/半年K(6M)/年K(1Y)；季/半年/年K 由日线按日历时间聚合，标签对齐周期起点
- 季K/半年K/年K 的**前复权必须在聚合重采样之前对日线完成**，否则拆股缺口会被混入大周期蜡烛
