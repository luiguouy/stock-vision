## 已完成功能

### 多市场 K 线浏览

- **实现位置**：`backend/data_provider.py` + `frontend/src/App.vue`
- **实现方式**：腾讯证券(主) + Yahoo Finance(备) 双源故障转移，支持 7 种周期（4h/1d/1w/1M/3M/6M/1Y），best-of 策略选最优数据源
- **完成时间**：2026-07-04（MVP）
- **依赖**：无

### 支撑/阻力位自动识别

- **实现位置**：`backend/analysis.py` + `frontend/src/utils/mockEngine.ts`
- **实现方式**：scipy.signal.argrelextrema 检测极值 → 时间衰减加权 → ATR 自适应聚类 → 排序取 Top-N，前后端双实现
- **完成时间**：2026-07-04
- **依赖**：多市场 K 线浏览

### 区间涨跌幅统计

- **实现位置**：`backend/analysis.py` + `frontend/src/utils/mockEngine.ts`
- **实现方式**：在指定日期区间内遍历所有子区间，计算最大连续涨幅和最大连续跌幅
- **完成时间**：2026-07-04
- **依赖**：多市场 K 线浏览

### 股票模糊搜索

- **实现位置**：`backend/main.py`（/api/search）+ `frontend/src/App.vue`
- **实现方式**：本地 26 只股票池，支持名称/代码/拼音缩写匹配，全市场覆盖
- **完成时间**：2026-07-04
- **依赖**：无

### 浏览器式导航栏

- **实现位置**：`frontend/src/App.vue`（header 区域）
- **实现方式**：后退/前进/刷新按钮 + 搜索栏 + 下拉建议，navBackHistory/navForwardHistory 双栈实现，selectWatchlist 和 handleSearch 均推入历史
- **完成时间**：2026-07-15
- **依赖**：股票模糊搜索

### 自选分组管理

- **实现位置**：`backend/main.py`（/api/watchlist GET/POST）+ `frontend/src/App.vue`
- **实现方式**：后端 JSON 文件持久化，支持分组 CRUD（创建/重命名/删除），标的拖入分组，兼容旧版扁平 list 格式
- **完成时间**：2026-07-09
- **依赖**：无

### 实时行情引擎

- **实现位置**：`frontend/src/utils/quote.ts` + `backend/quote_provider.py` + `backend/quote_config.py`
- **实现方式**：多市场分层轮询（美股/A股/港股各市场独立间隔），三层限流保护（全局冷却 + 单标的间隔 + IP 滑动窗口），自适应冷门标的降频
- **完成时间**：2026-07-09
- **依赖**：配置集中管理（quote_config.py）

### SQLite K 线缓存

- **实现位置**：`backend/db_cache.py`
- **实现方式**：klines 表（逐根存储）+ meta 表（更新时间/数据源），24h TTL，读写双重校验
